## ๐ Descripciรณn del Proyecto

Sistema de informaciรณn web multicapa diseรฑado para gestionar el conocimiento generado en una universidad con sedes y seccionales en varias ciudades del paรญs. Este proyecto permite registrar, consultar y analizar informaciรณn acadรฉmica, de investigaciรณn y curricular para apoyar la toma de decisiones y el mejoramiento continuo.
